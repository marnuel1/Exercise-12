Exercise12

1. Create an index.html file and a style.css file in the appropriate folder. Attach styles to the HTML document. 2. Create 10 paragraphs in HTML, and then define their height and width as 100px in styles. 3. Set the background color of all paragraphs to red. 4. Then change the background color to blue for the three selected paragraphs - you can highlight them with the same value of the class attribute 5. Add transparency to the other two (opacity property) - you can highlight them with the same class attribute value
